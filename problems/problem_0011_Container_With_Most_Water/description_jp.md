# 11. Container With Most Water (最も多くの水を含む容器)

長さ n の整数配列 `height` が与えられます。n 本の垂直線が引かれており、i 番目の線の 2 つの端点は (i, 0) と (i, height[i]) です。

x 軸と共に容器を形成する 2 本の線を見つけ、その容器が最も多くの水を含むようにしてください。

容器が保持できる水の最大量を返してください。

注意：容器を傾けることはできません。

## 例 1:
![](https://s3-lc-upload.s3.amazonaws.com/uploads/2018/07/17/question_11.jpg)
入力: height = [1,8,6,2,5,4,8,3,7]
出力: 49
説明: 上記の垂直線は配列 [1,8,6,2,5,4,8,3,7] で表されます。この場合、容器が保持できる水の最大面積（青い部分）は 49 です。

## 例 2:
入力: height = [1,1]
出力: 1

## 制約:
- n == height.length
- 2 <= n <= 10^5
- 0 <= height[i] <= 10^4
