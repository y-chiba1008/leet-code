# 21. 2つのソート済みリストの連結
2つのソート済み連結リスト `list1` と `list2` の先頭（head）が与えられます。

これら2つのリストを1つのソート済みリストにマージしてください。このリストは、最初の2つのリストのノードを繋ぎ合わせることによって作成する必要があります。

マージされた連結リストの先頭を返してください。

## 例 1:
![](https://assets.leetcode.com/uploads/2020/10/03/merge_ex1.jpg)
入力: list1 = [1,2,4], list2 = [1,3,4]
出力: [1,1,2,3,4,4]

## 例 2:
入力: list1 = [], list2 = []
出力: []

## 例 3:
入力: list1 = [], list2 = [0]
出力: [0]

## 制約:
- 両方のリストのノード数は [0, 50] の範囲内です。
- -100 <= Node.val <= 100
- list1 と list2 はどちらも非減少（昇順）でソートされています。
