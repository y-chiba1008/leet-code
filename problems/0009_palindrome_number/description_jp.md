# 9. 回文数

整数 `x` が与えられたとき、`x` が回文であれば `true` を、そうでなければ `false` を返してください。

## 例 1:
入力: x = 121
出力: true
説明: 121 は左から読んでも右から読んでも 121 です。

## 例 2:
入力: x = -121
出力: false
説明: 左から読むと -121 です。右から読むと 121- になります。したがって、回文ではありません。

## 例 3:
入力: x = 10
出力: false
説明: 右から読むと 01 になります。したがって、回文ではありません。

## 制約:
-2^31 <= x <= 2^31 - 1
