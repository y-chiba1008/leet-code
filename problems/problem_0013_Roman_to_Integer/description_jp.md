# 13. ローマ数字から整数へ (Roman to Integer)

ローマ数字は、以下の 7 つの異なる記号で表されます: I, V, X, L, C, D, M。

| 記号 | 値   |
| ---- | ---- |
| I    | 1    |
| V    | 5    |
| X    | 10   |
| L    | 50   |
| C    | 100  |
| D    | 500  |
| M    | 1000 |

例えば、2 はローマ数字で II と書かれ、1 を 2 つ足したものです。12 は XII と書かれ、これは単に X + II です。27 は XXVII と書かれ、XX + V + II です。

ローマ数字は通常、左から右に大きいものから小さいものへと書かれます。しかし、4 の数字は IIII ではありません。代わりに、4 は IV と書かれます。1 が 5 の前にあるため、それを引いて 4 になります。同じ原理が 9 にも適用され、IX と書かれます。引き算が使用される例は以下の 6 つです：

- I は V (5) と X (10) の前に置くことができ、4 と 9 を作ります。
- X は L (50) と C (100) の前に置くことができ、40 と 90 を作ります。
- C は D (500) と M (1000) の前に置くことができ、400 と 900 を作ります。

ローマ数字が与えられたとき、それを整数に変換してください。

## 例 1:
入力: s = "III"
出力: 3
説明: III = 3.

## 例 2:
入力: s = "LVIII"
出力: 58
説明: L = 50, V= 5, III = 3.

## 例 3:
入力: s = "MCMXCIV"
出力: 1994
説明: M = 1000, CM = 900, XC = 90, IV = 4.

## 制約:
- 1 <= s.length <= 15
- s は記号 ('I', 'V', 'X', 'L', 'C', 'D', 'M') のみを含む。
- s は [1, 3999] の範囲の有効なローマ数字であることが保証されている。
